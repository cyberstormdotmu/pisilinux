Texlivemodules¶

Author:Semen Cirit
Date:October 26, 2011
Version:0.1

compile¶

compile(parameters)

Compiles texlive packages with given parameters. It first moves tlpobj sources, then it generatea configure files and builds format files.

install¶

install(parameters)

Installs texlive packages with given parameters. It fist creates symlinks for the formats. Then it installs doc, config and texmf files to the system.

createSymlinksFormat2Engines¶

createSymlinksFormat2Engines()

Creates symlinks for the engine names in /usr/texmf/fmtutil/format*.cnf files.

addFormat¶

addFormat()

Adds new formats created via new config files.

moveSources¶

moveSources()

Moves unneccessary sources under /usr/share/tlpkg/tlpobj/ files.

buildFormatFiles¶

buildFormatFiles()

Builds format files with running fmutil.

generateConfigFiles¶

generateConfigFiles()

makeLanguagesDefDatLines¶

makeLanguagesDefDatLines()

installDocFiles()
