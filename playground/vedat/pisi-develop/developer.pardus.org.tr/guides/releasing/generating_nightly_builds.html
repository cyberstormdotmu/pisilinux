Generating Nightly Builds¶

Author:Semen Cirit

Last Modificaiton Date: October 26, 2011

Version:0.1

Nightly builds are the daily test releases of pardus. In order to automatically generate these nightly builds and upload them to ftp servers the following steps are performed:

  1. Check out nightly script from core source code repository:
    svn co http://svn.pardus.org.tr/uludag/trunk/scripts/nightly/
  2. Add the script to /etc/crontab and start nightly script as a cron job:
    15 3  * * * root    /root/nightly/build.sh > /root/nightly/build-all.log 2>&1
