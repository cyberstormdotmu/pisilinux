COMAR (Configuration Manager)¶

COMAR is COnfiguration MAnageR of Pardus Linux, which is a DBus service that helps the installed software operate flawlessly. COMAR knows the tasks that can be provided by each application, together with the functionality they depend on and other information. Different applications may adapt themselves according to the presence and capability of their peers.

Download¶

Latest stable release is 2.0.5. You can also fetch code from SVN repository.

COMAR also has a development release, and can be downloaded from development branch.
